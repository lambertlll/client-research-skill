# 客户尽调助手 (Client Research Skill)

OpenClaw 技能 - 为银行从业人员提供快速的客户背景调查和访前准备

## ✨ 核心功能

✅ 智能识别企业/个人客户类型  
✅ 自动抓取公开信息（工商、财报、新闻、社交媒体）  
✅ 重点突出近3个月最新动态  
✅ 生成结构化报告（基本信息、最新动态、风险提示、访前建议）  
✅ 提供业务机会分析和话题切入建议

## 📝 使用示例

```
帮我查一下比亚迪
帮我查一下雷军
帮我查一下王传福和比亚迪
```
