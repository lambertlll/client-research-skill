# 客户尽调助手 - README

## 简介

这是一个为银行从业人员设计的客户尽调技能，通过抓取公开信息，快速生成结构化的客户背景报告，帮助客户经理在拜访前充分了解客户情况。

## 功能特点

✅ **智能识别** - 自动判断企业/个人客户类型  
✅ **全面覆盖** - 基本信息、最新动态、风险提示、访前建议  
✅ **纯公开信息** - 无需接入内部系统，合规安全  
✅ **结构化输出** - 统一格式，方便快速浏览  
✅ **时效性强** - 重点抓取近3个月最新信息

## 支持的客户类型

### 企业客户
- 工商信息、股权结构、经营情况
- 核心团队、财务数据、市场地位
- 最新动态（业绩、合作、管理层发言）
- 风险提示（诉讼、失信、处罚）
- 行业分析、访前建议

### 个人客户
- 职业背景、职业成就、关联企业
- 最新动态（公开发言、社交媒体、行业活动）
- 风险提示（失信、涉诉、负面舆情）
- 访前建议

## 数据源

### 企业信息
- 国家企业信用信息公示系统
- 上市公司公告（巨潮资讯网、交易所官网）
- 天眼查/企查查（通过搜索引擎）

### 风险信息
- 中国裁判文书网（诉讼）
- 信用中国（失信记录）
- 行政处罚公示

### 新闻舆情
- 财经媒体（财新、财经、证券时报等）
- 科技媒体（36氪、虎嗅、钛媒体）
- 社交媒体（微博、知乎）

### 个人信息
- LinkedIn、微博、个人公众号
- 行业媒体人物专访
- 论坛演讲、峰会报道

## 报告内容

### 企业报告包含
1. 基本信息（工商、注册、经营范围）
2. 核心团队（董事长、CEO、高管）
3. 股权结构（实际控制人、主要股东）
4. 经营情况（财务数据、市场地位）
5. **最新动态**（业绩、合作、管理层发言、舆情）⭐
6. 风险提示（诉讼、失信、处罚）
7. 行业分析（政策、趋势、竞争格局）
8. 访前建议（关注要点、话题切入、业务机会）

### 个人报告包含
1. 基本信息（职业背景、教育经历）
2. 职业履历（现任、历任职务）
3. 职业成就（影响力、获奖、社会职务）
4. 关联企业（任职、投资、兼任）
5. **最新动态**（公开发言、社交媒体、行业活动）⭐
6. 风险提示（失信、涉诉、负面舆情）
7. 行业背景（所在行业、专业领域）
8. 访前建议（关注要点、话题切入、业务机会）

## 注意事项

### 合规要求
✅ 所有数据来自公开渠道  
✅ 不存储客户真实信息  
✅ 报告仅供内部参考，不得外传  
✅ 实际业务以官方渠道核实为准

### 数据时效性
- 优先抓取近3个月信息
- 重大事项可追溯6个月
- 建议每月更新重点客户报告

### 风险提示
- 公开信息可能存在滞后
- 负面信息需进一步核实
- 涉及授信决策需结合内部数据

## 常见问题

**Q: 报告生成需要多长时间？**  
A: 通常2-3分钟，取决于客户的公开信息量。

**Q: 可以查询哪些国家/地区的企业？**  
A: 主要支持中国大陆企业，港股、美股上市公司也可查询。

**Q: 个人客户必须是企业高管吗？**  
A: 不一定，但需要有一定公开信息（如媒体报道、社交媒体等）。

**Q: 报告的准确性如何保证？**  
A: 所有信息来自公开渠道，并标注来源。建议结合内部数据交叉验证。

**Q: 可以定期更新重点客户吗？**  
A: 当前版本需手动触发，v1.2版本将支持自动定期更新。
