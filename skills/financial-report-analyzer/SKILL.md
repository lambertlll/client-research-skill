# 财报分析助手

## 定位
从**银行信贷/授信视角**分析上市公司财务报告，评估企业偿债能力和信用风险，为授信决策和领导汇报提供专业财务分析报告。

> 银行视角的核心问题只有一个：**这家公司能不能还钱？为什么？**

## 触发条件
- "帮我做 XX 的财报分析"
- "分析 XX 的财务状况"
- "XX 的财务分析报告"
- "做一份 XX 的财报分析"
- "帮我看看 XX 的财务数据"

触发后**主动询问用户**：
> "需要简化版还是详细版？
> - 简化版：快速出报告，基于公开财经媒体数据，适合快速了解
> - 详细版：深度解读年报原文，分析每项指标的构成与变动原因，适合正式汇报和授信决策"

## 输出格式
- **默认：Word 文档（.docx）**，通过 Telegram 发送
- 用户明确说 Markdown/md 时输出纯文本
- 转换命令：`pandoc "文件名.md" -o "文件名.docx" --from markdown --to docx`

## 简化版流程

**适用场景：** 快速了解公司基本财务状况，时间有限，数据来源为财经媒体摘要。

### 数据获取
1. session_status 确认当前日期，确定最新财报期
2. web_search 搜索财经媒体报道获取核心财务数据
3. web_search 搜索近期热点事件、战略动向
4. 整理指标，生成报告

### 报告结构（简化版）
```
一、公司基本情况
二、核心财务指标速览（三年对比+趋势判断）
三、公司动态与战略分析
四、偿债能力分析
五、现金流质量分析
六、盈利能力分析
七、资产质量分析
八、负债结构与融资分析
九、经营风险分析
十、综合信用评估与授信建议
附：主要财务数据汇总表
```

## 详细版流程

**适用场景：** 正式授信材料、领导汇报，需深度解读年报原文，分析每项指标的具体构成与变动原因。

### 第一步：获取原始年报PDF

**优先级：**
1. 港交所披露易（hkexnews.hk）— 港股
2. 巨潮资讯（cninfo.com.cn）— A股
3. 上交所（sse.com.cn）/ 深交所（szse.cn）— A股
4. 公司官网投资者关系页面

**搜索策略：**
```
[公司名] 2025年报 annual report PDF site:hkexnews.hk
[公司名] 年度报告 2025 site:cninfo.com.cn
[股票代码] 年报 filetype:pdf
```

**PDF解析：** 使用 `pdf` 工具读取年报，重点章节：
- 财务摘要（Financial Highlights）
- 管理层讨论与分析（MD&A）
- 合并资产负债表
- 合并利润表
- 合并现金流量表
- 财务报表附注（Notes）— 负债明细、关联交易、或有负债

### 第二步：外部搜索补充
- 近期重大事件：`[公司名] 2025 2026 战略 并购 公告`
- 管理层解读：`[公司名] 业绩会 管理层 电话会 解读`
- 行业政策：`[行业] 政策 监管 2025 2026`
- 风险信息：`[公司名] 诉讼 质押 违约 风险`
- 券商研报：`[公司名] 研报 评级 2026`

### 第三步：逐项深度分析

对每个财务指标，必须回答三个问题：
1. **数字是多少？**（具体数值，三年对比）
2. **由什么构成？**（拆分到业务线/科目/地区）
3. **为什么变动？**（驱动因素，来自MD&A或管理层解读）

### 报告结构（详细版）
```
一、公司基本情况
二、核心财务指标速览
三、公司动态与战略分析
  - 近期重大事件（含影响评估）
  - 战略方向与管理层表态
  - 行业与政策环境
四、偿债能力深度分析
  4.1 短期偿债：流动比率/速动比率/现金比率 + 构成说明
  4.2 长期偿债：资产负债率/利息保障倍数 + 变动原因
  4.3 债务结构明细：短期/长期/债券/非标 + 到期安排
  4.4 融资渠道与授信情况
五、现金流深度分析
  5.1 经营现金流：主要来源与去向，与净利润背离分析
  5.2 投资现金流：资本开支明细，投资方向
  5.3 筹资现金流：借款/还款/分红/回购明细
  5.4 自由现金流趋势与短债覆盖能力
六、盈利能力深度分析
  6.1 收入拆分：各业务线/地区/产品收入及增速原因
  6.2 成本结构：各类成本占比变化及原因
  6.3 毛利率变化：业务结构变化 vs 定价/成本影响
  6.4 净利率：非经常性项目影响，扣非利润分析
七、资产质量深度分析
  7.1 应收账款：规模、增速、账龄结构、坏账准备
  7.2 存货：规模、周转率、跌价风险
  7.3 商誉与无形资产：来源、减值测试结果
  7.4 长期投资：主要被投企业及公允价值变动
八、负债结构深度分析
  8.1 经营性负债 vs 金融性负债拆分
  8.2 有息负债明细：银行借款/债券/租赁负债/其他
  8.3 债务到期结构（一年内/1-3年/3年以上）
  8.4 融资成本：平均利率，与同业比较
  8.5 对外担保与质押情况
九、经营风险分析
  - 客户/供应商集中度（附具体数据）
  - 关联交易明细
  - 重大诉讼与或有负债
  - 股权质押比例
  - 行业政策风险量化
十、综合信用评估与授信建议
  - 信用评级及评级依据（数据支撑）
  - 主要风险点（量化）
  - 授信建议（额度/期限/担保/用途）
  - 重点监测指标与预警阈值
附：完整三张报表数据（资产负债表/利润表/现金流量表）
```

## 分析框架（银行信贷视角）

### 偿债能力指标参考标准
| 指标 | 优秀 | 良好 | 警戒 |
|------|------|------|------|
| 流动比率 | >2.0 | 1.5-2.0 | <1.0 |
| 速动比率 | >1.5 | 1.0-1.5 | <0.8 |
| 资产负债率 | <40% | 40-60% | >70% |
| 利息保障倍数 | >5x | 3-5x | <2x |
| 经营现金流/短债 | >1.0 | 0.5-1.0 | <0.3 |

### 行业特定关注点
**制造业：** 产能利用率、存货周转、应收账款质量、原材料价格
**房地产：** 三道红线、土储、预售、区域布局
**科技互联网：** 用户规模、变现率、研发投入、AI布局
**零售/消费：** 同店增长、库存、渠道扩张
**金融：** 不良率、拨备覆盖率、资本充足率

## 数据源优先级
1. 港交所披露易 / 巨潮资讯（原始年报PDF）
2. 公司官网投资者关系页面
3. 上交所/深交所公告
4. 新浪财经、东方财富（结构化财务数据）
5. 财经媒体（财新、证券时报、21经济网）

## 注意事项
- 所有数据注明来源和时间
- ⚠️ 标注异常指标
- 详细版每项指标必须说明变动原因，不能只列数字
- 报告措辞专业、客观，适合正式汇报场景
- 授信建议仅供参考，最终决策需结合银行内部规定
- 非上市企业暂不支持（无公开财报）
