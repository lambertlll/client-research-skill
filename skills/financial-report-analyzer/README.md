# 财报分析助手

> **银行信贷视角**的上市公司财报分析工具，自动生成可直接用于汇报的 Word 分析报告。

---

## 这是什么

一个专门为**银行从业人员**设计的财报分析 Skill。

和投资分析不同，银行视角只关心一个核心问题：**这家公司能不能还钱？**

输入公司名称，输出一份结构完整、措辞专业的授信分析报告，可直接用于：
- 授信评审材料
- 领导汇报
- 客户拜访前背景了解

---

## 快速开始

```
帮我做腾讯的财报分析
```

Skill 会先询问你需要哪种深度：

| | 简化版 | 详细版 |
|---|---|---|
| 数据来源 | 财经媒体摘要 | 年报原文 PDF |
| 分析深度 | 关键指标 + 方向判断 | 每项指标构成 + 变动原因 |
| 适合场景 | 快速了解、初步判断 | 正式授信材料、领导汇报 |
| 生成时间 | 约 5-10 分钟 | 约 20-40 分钟 |
| 报告章节 | 7 章 | 10 章 + 附表 |

---

## 报告内容

### 简化版（7章）

1. 公司基本情况
2. 核心财务指标速览
3. 公司动态与战略分析
4. 偿债能力分析
5. 现金流质量分析
6. 盈利能力分析
7. 综合信用评估与授信建议

### 详细版（10章 + 附）

在简化版基础上深化：
- **每项指标**提供构成拆分 + 变动原因（引用 MD&A）
- **现金流**拆分经营/投资/筹资三类明细
- **负债结构**列明有息负债类型、到期安排、平均利率
- **风险分析**量化支撑，含评分矩阵
- **附表**：完整三张报表 + 主要比率汇总

---

## 数据来源

按优先级：

1. 港交所披露易 / 巨潮资讯（原始年报 PDF）
2. 公司官网投资者关系页面
3. 上交所 / 深交所公告
4. 新浪财经、东方财富（补充结构化数据）

---

## 输出格式

默认生成 **Word 文档（.docx）**，发送至 Telegram。

---

## 注意事项

- 仅支持**上市公司**（需有公开财报）
- 详细版依赖年报 PDF 可获取性，部分公司可能需要手动提供 PDF 链接
- 授信建议仅供参考，最终决策需遵循银行内部合规流程
- 所有数据标注来源与时间
