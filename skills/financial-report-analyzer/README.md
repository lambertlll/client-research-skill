# 财报分析助手 - README

## 简介

这是一个为银行从业人员设计的财报分析技能，从**信贷授信视角**分析上市公司财务报告，自动生成结构化的 Word 分析报告，可直接用于授信评审、领导汇报和客户拜访准备。

## 功能特点

✅ **信贷视角** - 聚焦偿债能力、现金流质量、负债结构  
✅ **双模式** - 简化版快速判断，详细版深度解读  
✅ **自动化** - 输入公司名称，输出完整 Word 报告  
✅ **专业措辞** - 符合银行内部汇报规范  
✅ **数据溯源** - 所有数据标注来源与时间

## 核心问题

和投资分析不同，银行视角只关心一个核心问题：**这家公司能不能还钱？**

重点关注：
- 偿债能力（流动比率、速动比率、资产负债率）
- 现金流质量（经营现金流净额、自由现金流）
- 负债结构（有息负债、短期偿债压力、融资成本）
- 经营风险（应收账款、存货周转、关联交易）

## 分析模式

| | 简化版 | 详细版 |
|---|---|---|
| 数据来源 | 财经媒体摘要 | 年报原文 PDF |
| 分析深度 | 关键指标 + 方向判断 | 每项指标构成 + 变动原因 |
| 适合场景 | 快速了解、初步判断 | 正式授信材料、领导汇报 |
| 生成时间 | 约 5-10 分钟 | 约 20-40 分钟 |
| 报告章节 | 7 章 | 10 章 + 附表 |

## 报告内容

### 简化版（7章）

1. **公司基本情况** - 主营业务、行业地位、股权结构
2. **核心财务指标速览** - 关键指标汇总表
3. **公司动态与战略分析** - 最新业绩、战略调整、市场动态
4. **偿债能力分析** - 流动比率、速动比率、资产负债率
5. **现金流质量分析** - 经营现金流、自由现金流、现金流量比率
6. **盈利能力分析** - 营收、净利润、毛利率、ROE
7. **综合信用评估与授信建议** - 信用评分、授信建议、风险提示

### 详细版（10章 + 附表）

在简化版基础上深化：
- **每项指标**提供构成拆分 + 变动原因（引用 MD&A）
- **现金流**拆分经营/投资/筹资三类明细
- **负债结构**列明有息负债类型、到期安排、平均利率
- **风险分析**量化支撑，含评分矩阵
- **附表**：完整三张报表 + 主要比率汇总

## 数据来源

### 优先级顺序

1. **原始年报 PDF**（详细版必需）
   - 港交所披露易
   - 巨潮资讯网
   - 上交所/深交所公告

2. **公司官网**
   - 投资者关系页面
   - 财报下载区

3. **财经媒体**（简化版可用）
   - 新浪财经
   - 东方财富
   - 每经网、21经济网

### 搜索策略

- 关键词：`[公司名] 2025年报 全年 财报 2026年3月`
- 优先选择：官方披露 > 财经媒体 > 第三方平台
- 时效性：优先最新一期年报/季报

## 注意事项

### 适用范围
✅ 支持**上市公司**（A股、港股、美股）  
✅ 需有公开财报（年报/季报）  
❌ 不支持非上市公司、拟上市公司

### 数据时效性
- 优先使用最新一期年报/季报
- 详细版依赖年报 PDF 可获取性
- 部分公司可能需要手动提供 PDF 链接

### 合规要求
✅ 所有数据来自公开渠道  
✅ 授信建议仅供参考，最终决策需遵循银行内部合规流程  
✅ 报告仅供内部使用，不得外传  
✅ 实际授信以官方审批为准

### 风险提示
- 财报数据可能存在滞后
- 需结合行业政策、宏观环境综合判断
- 重大事项需进一步核实
- 涉及授信决策需结合内部评级系统

## 常见问题

**Q: 报告生成需要多长时间？**  
A: 简化版约 5-10 分钟，详细版约 20-40 分钟，取决于财报数据的可获取性。

**Q: 可以分析哪些国家/地区的公司？**  
A: 主要支持中国大陆上市公司（A股），港股、美股也可分析。

**Q: 详细版和简化版有什么区别？**  
A: 详细版基于年报 PDF 原文，提供每项指标的构成拆分和变动原因；简化版基于财经媒体摘要，快速判断财务状况。

**Q: 报告的准确性如何保证？**  
A: 所有数据来自公开渠道，并标注来源。建议结合银行内部评级系统交叉验证。

**Q: 可以定期更新重点客户吗？**  
A: 当前版本需手动触发，v1.3 版本将支持自动定期更新。

**Q: 授信建议的依据是什么？**  
A: 基于偿债能力、现金流质量、盈利能力、行业地位等多维度综合评估，仅供参考。
