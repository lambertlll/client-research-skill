# 财报分析助手 (Financial Report Analyzer)

银行信贷视角的上市公司财务分析工具，自动生成 Word 格式财报分析报告。

## 定位

从银行信贷/授信视角分析上市公司财务报告，评估企业偿债能力和信用风险，为授信决策和领导汇报提供专业财务分析报告。

> 区别于投资分析（关注成长性和估值），银行视角的核心问题：**这家公司能不能还钱？**

## 两种模式

### 简化版
- 基于公开财经媒体数据快速出报告
- 适合时间有限的快速了解场景
- 约 30 分钟内完成

### 详细版
- 深度解读年报原文 PDF，分析每项指标的构成与变动原因
- 适合正式授信材料、领导汇报
- 约 1-2 小时完成

## 触发条件

触发后会询问需要哪种模式：

```
需要简化版还是详细版？
- 简化版：快速出报告，基于公开财经媒体数据
- 详细版：深度解读年报原文，适合正式汇报和授信决策
```

常用触发词：
- "帮我做 XX 的财报分析"
- "分析 XX 的财务状况"
- "XX 的财务分析报告"
- "做一份 XX 的财报分析"

## 输出格式

- **默认：Word 文档（.docx）**，通过 Telegram 发送
- 可选：Markdown（需明确说明）

## 分析维度

### 简化版（7章）
1. 公司基本情况
2. 核心财务指标速览
3. 公司动态与战略分析
4. 偿债能力分析
5. 现金流质量分析
6. 盈利能力分析
7. 综合信用评估与授信建议

### 详细版（10章 + 附）
1. 公司基本情况
2. 核心财务指标速览
3. 公司动态与战略分析
4. 偿债能力深度分析（含债务结构明细）
5. 现金流深度分析（经营/投资/筹资分项）
6. 盈利能力深度分析（收入拆分+成本+毛利率原因）
7. 资产质量分析
8. 负债结构深度分析
9. 经营风险分析（量化风险矩阵）
10. 综合信用评估与授信建议（含评分矩阵）

## 银行信贷视角核心指标

| 指标 | 优秀 | 良好 | 警戒 |
|------|------|------|------|
| 流动比率 | >2.0 | 1.5-2.0 | <1.0 |
| 速动比率 | >1.5 | 1.0-1.5 | <0.8 |
| 资产负债率 | <40% | 40-60% | >70% |
| 利息保障倍数 | >5x | 3-5x | <2x |
| 经营现金流/短债 | >1.0 | 0.5-1.0 | <0.3 |

## 数据来源

### 简化版
- 新浪财经、东方财富（结构化数据）
- 财经媒体（财新、证券时报）
- 券商研报

### 详细版
- 港交所披露易 / 巨潮资讯（原始年报 PDF）
- 公司官网投资者关系页面
- 上交所/深交所公告
- 管理层讨论与分析（MD&A）

## 使用示例

```
用户：帮我做腾讯的财报分析

助手：需要简化版还是详细版？
- 简化版：快速出报告
- 详细版：深度解读年报

用户：详细版

助手：[执行年报数据获取 + 分析 + 生成报告...]
[生成 Word 文档并发送]
```

## 注意事项

- 所有数据注明来源和时间
- 详细版依赖年报 PDF 可获取性
- 报告措辞专业、客观，适合正式汇报场景
- 授信建议仅供参考，最终决策需结合银行内部规定
- 非上市企业暂不支持（无公开财报）
