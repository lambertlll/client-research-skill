### 2. 报告核心内容

#### ⭐⭐⭐ 最新变化速览（重点）
- **财务指标最新变化** - 环比/同比对比，异常预警
- **战略调整最新动态** - 重大投资、业务转型、管理层变动
- **行业环境最新变化** - 政策、竞争、上下游

#### ⭐⭐⭐ 审贷会提问预测（重点）
- **最新变化类**（5-8个）- 基于财务异常
- **战略调整类**（3-5个）- 基于战略变化
- **行业环境类**（3-5个）- 基于行业动态
- **常规财务类**（2-3个）- 基于持续风险

每个提问包含：
- 数据支撑
- 可能原因分析
- 建议回答
- 需要准备的材料

#### ⭐⭐ 行业深度分析
- 行业最新政策
- 行业景气度
- 竞争格局变化
- 上下游动态
- 行业专家观点

#### ⭐⭐ 客户战略分析
- 战略调整
- 重大投资
- 市场拓展
- 管理层变动

#### 财务数据深度分析
- 三年财务数据对比
- 关键财务指标
- 财务趋势分析
- 同行业对比

### 4. 报告亮点

#### ⭐ 最新变化优先
报告第一部分就是**最新变化速览**，快速了解企业近期动态：
- 财务指标环比/同比变化
- 异常指标预警
- 战略调整动态
- 行业环境变化

#### ⭐ 提问针对性强
基于最新变化生成提问，而非泛泛而谈：
- "为什么Q3营收下滑15%，而竞争对手都在增长？"
- "短期借款从20亿激增至50亿，资金用途是什么？"
- "进军新能源领域的战略考虑是什么？风险如何控制？"

#### ⭐ 建议回答实用
每个提问都提供：
- 数据支撑（具体数字）
- 可能原因分析（3-5个原因）
- 建议回答要点（可直接使用）
- 需要准备的材料清单

#### ⭐ 行业分析深入
专门章节分析行业动态：
- 最新政策及影响
- 行业景气度判断
- 竞争格局变化
- 上下游动态
- 券商研报观点

### 5. 提问预测示例

**最新变化类提问：**
```
Q: 为什么Q3营收下滑15%，而竞争对手都在增长？

数据支撑：
- Q3营收85亿，同比-15%，环比-8%
- 行业平均增长+12%
- 主要竞争对手A增长+18%，B增长+10%

建议回答：
"Q3营收下滑主要有三个原因：
1. 公司主动优化客户结构，放弃了部分毛利率低于15%的订单
2. 两个大客户的订单从Q3延期至Q4
3. 新产能将在Q4投产，预计明年营收将恢复双位数增长"

需要准备的材料：
- 客户结构调整明细
- 延期订单合同
- 新产能投产计划
- Q4业绩预测
```

**战略调整类提问：**
```
Q: 进军新能源领域的战略考虑是什么？风险如何控制？

背景：
- 2026年2月宣布投资50亿进军新能源
- 传统业务增长放缓
- 行业竞争加剧

建议回答：
[根据公司战略填写]
```

### 6. 行业特点关注

根据不同行业，关注特定指标：

**制造业：**
- 产能利用率、存货周转、应收账款质量

**房地产：**
- 土地储备、预售情况、三道红线指标

**科技企业：**
- 研发投入、技术壁垒、客户集中度

**零售/消费：**
- 同店增长、库存周转、渠道扩张

### 7. 注意事项

✅ **可以做的：**
- 查询任何上市公司
- 重复查询（获取最新信息）
- 保存报告供审贷会使用

⚠️ **注意：**
- 报告仅供参考，不能替代尽职调查
- 提问预测准确率约70-80%，需结合实际调整
- 建议审贷会前1周内更新报告

❌ **不能做的：**
- v1.0不支持非上市企业（需v2.0）
- 不能替代内部评审流程
- 不能作为唯一的授信决策依据
