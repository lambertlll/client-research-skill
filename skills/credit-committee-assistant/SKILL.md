# 审贷会助手

## 描述
为银行客户经理提供授信审批会（审贷会）前的准备工具。通过分析上市公司的财务数据、战略调整、行业动态和最新变化，预测审贷会可能的提问并提供建议回答，帮助客户经理充分准备审贷会。

## 触发条件
用户提到以下场景时激活此技能：
- "准备审贷会"
- "审批会准备"
- "上会材料"
- "帮我准备 [公司名] 的审贷会"
- "分析 [公司名] 的审贷会提问"
- 明确提到需要准备授信审批会、信贷审批会、审贷委员会等

## 输出格式
- **默认格式：** Markdown（.md）
- **Word格式：** 当用户明确要求"Word"、"word"、"docx"、"发送Word"、"生成Word格式"时，自动转换为Word文档（.docx）并发送
- **转换工具：** 使用pandoc将Markdown转换为Word格式

## 核心功能

### 1. 最新变化分析（重点）⭐⭐⭐
- **财务指标最新变化**
  - 营收、净利润、毛利率、现金流等关键指标的环比/同比变化
  - 异常指标预警（如营收下滑、现金流恶化、负债激增）
  - 财务结构变化（如资产负债率、流动比率）

- **战略调整最新动态**
  - 业务转型、多元化、聚焦主业
  - 重大投资、并购、资本开支计划
  - 市场拓展（新市场、新产品、新客户）
  - 管理层变动及影响

- **行业环境最新变化**
  - 最新行业政策及影响
  - 行业景气度变化（供需、价格）
  - 竞争格局变化
  - 上下游动态

### 2. 审贷会提问预测（重点）⭐⭐⭐
基于最新变化和异常指标，预测审贷会可能的提问，并提供建议回答。

**提问分类：**
- **最新变化类**（优先级最高）
  - 针对最近1个季度的财务异常
  - 针对最新战略调整
  - 针对最新行业变化

- **战略调整类**
  - 业务转型的合理性和风险
  - 重大投资的回报预期
  - 市场拓展的可行性

- **行业环境类**
  - 行业政策影响
  - 行业竞争态势
  - 上下游风险

- **常规财务类**
  - 偿债能力
  - 盈利能力
  - 营运能力

**每个提问包含：**
- 数据支撑（具体数字、对比）
- 可能原因分析
- 建议回答要点
- 需要准备的补充材料

### 3. 财务数据深度分析
- **三年财务数据对比**
  - 资产负债表：总资产、净资产、负债总额、资产负债率
  - 利润表：营收、毛利率、净利润、净利率
  - 现金流量表：经营性现金流、投资性现金流、筹资性现金流

- **关键财务指标**
  - 偿债能力：流动比率、速动比率、资产负债率、利息保障倍数
  - 盈利能力：毛利率、净利率、ROE、ROA
  - 营运能力：应收账款周转率、存货周转率、总资产周转率
  - 成长能力：营收增长率、净利润增长率

- **财务趋势分析**
  - 指标变化趋势（上升/下降/波动）
  - 异常指标预警
  - 同行业对比

### 4. 行业深度分析（重点）⭐⭐
- **行业最新政策**
  - 监管政策、产业政策
  - 政策对企业的影响分析

- **行业景气度分析**
  - 行业规模、增长率
  - 供需关系、价格走势
  - 行业周期判断

- **竞争格局变化**
  - 主要竞争对手动态
  - 市场份额变化
  - 技术革新、新进入者

- **上下游动态**
  - 供应链变化
  - 客户需求变化
  - 原材料价格波动

- **行业专家观点**
  - 券商研报
  - 行业分析师观点
  - 行业协会报告

### 5. 客户战略分析（重点）⭐⭐
- **战略调整**
  - 业务转型方向
  - 多元化 vs 聚焦主业
  - 战略目标和时间表

- **重大投资**
  - 新项目投资
  - 并购整合
  - 资本开支计划

- **市场拓展**
  - 新市场开拓
  - 新产品推出
  - 新客户开发

- **管理层变动**
  - 核心高管变化
  - 变动原因和影响

### 6. 行业特点关注
根据不同行业特点，关注特定指标和风险：

**制造业：**
- 产能利用率
- 存货周转
- 应收账款质量
- 原材料价格波动

**房地产：**
- 土地储备
- 预售情况
- 三道红线指标
- 区域布局

**科技企业：**
- 研发投入
- 技术壁垒
- 客户集中度
- 知识产权

**零售/消费：**
- 同店增长
- 库存周转
- 渠道扩张
- 品牌力

**金融：**
- 不良率
- 拨备覆盖率
- 资本充足率
- 净息差

## 工作流程

### 步骤1：识别企业
```
输入："帮我准备比亚迪的审贷会"
识别：比亚迪股份有限公司（002594.SZ）
```

### 步骤2：数据抓取
1. 搜索最新财报（最近3年 + 最新季报）
2. 搜索业绩发布会信息
3. 搜索最新新闻（近3个月）
4. 搜索行业报道和分析
5. 搜索竞争对手信息
6. 搜索券商研报

### 步骤3：数据分析
1. 计算财务指标（三年对比）
2. 识别异常指标和最新变化
3. 分析战略调整
4. 分析行业环境
5. 同行业对比

### 步骤4：提问生成
基于以下逻辑生成提问：
1. **最新变化优先** - 最近1个季度的异常变化
2. **战略调整** - 重大战略变化
3. **行业环境** - 行业政策、竞争格局变化
4. **常规财务** - 持续性风险点

### 步骤5：生成报告
使用报告模板生成结构化的审贷会准备材料

### 步骤6：格式转换（如需要）
如果用户要求Word格式：
1. 先生成Markdown文档
2. 使用pandoc转换为Word格式：
   ```bash
   pandoc "文件名.md" -o "文件名.docx" --from markdown --to docx
   ```
3. 通过message工具发送Word文档给用户

## 注意事项

### 适用范围
- ✅ v1.0主要针对A股、港股、美股上市公司
- ✅ 可获取公开财务数据的企业
- ⚠️ 非上市企业需要v2.0接入银行内部系统

### 数据时效性
- 优先使用最新季报/年报
- 重点关注近3个月的变化
- 行业数据尽量使用最新报告

### 提问预测准确性
- 基于财务异常和行业特点生成
- 需要结合具体企业情况调整
- 建议准备额外的应对方案

### 合规要求
- 所有数据来自公开渠道
- 报告仅供内部使用
- 实际授信决策需结合内部评审

// 异常检测
function detectAnomalies(ratios, trends) {
  const anomalies = [];
  
  // 检测营收下滑
  if (trends.revenueGrowth < -10%) {
    anomalies.push({
      type: '营收下滑',
      severity: 'high',
      question: '为什么营收下滑超过10%？'
    });
  }
  
  // 检测现金流恶化
  if (ratios.ocf < 0 && ratios.netIncome > 0) {
    anomalies.push({
      type: '现金流与利润背离',
      severity: 'high',
      question: '为什么有利润但现金流为负？'
    });
  }

// 提问生成
function generateQuestions(anomalies, strategy, industry) {
  const questions = [];
