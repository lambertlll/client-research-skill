# 审贷会助手 - README

## 简介

审贷会助手是为银行客户经理设计的授信审批会（审贷会）准备工具。通过分析上市公司的财务数据、战略调整、行业动态和最新变化，预测审贷会可能的提问并提供建议回答，帮助客户经理充分准备审贷会。

## 功能特点

✅ **最新变化分析** - 重点关注近3个月的财务、战略、行业变化  
✅ **审贷会提问预测** - 基于异常指标和最新变化生成提问  
✅ **建议回答** - 为每个提问提供数据支撑和回答要点  
✅ **财务深度分析** - 三年财务数据对比和关键指标分析  
✅ **行业深度分析** - 行业政策、竞争格局、上下游动态  
✅ **客户战略分析** - 战略调整、重大投资、市场拓展  
✅ **行业特点关注** - 根据不同行业关注特定指标

## 支持的企业类型

### 九、会前准备清单

## 提问预测逻辑

### 优先级1：最新变化类 ⭐⭐⭐
基于最近1个季度的异常变化：
- 营收/利润大幅波动
- 现金流异常
- 负债激增
- 毛利率大幅变化

### 优先级2：战略调整类 ⭐⭐
基于最新战略变化：
- 业务转型
- 重大投资/并购
- 市场拓展
- 管理层变动

### 优先级3：行业环境类 ⭐⭐
基于行业最新变化：
- 行业政策调整
- 竞争格局变化
- 上下游动态

### 优先级4：常规财务类 ⭐
基于持续性风险：
- 资产负债率
- 应收账款质量
- 存货周转

## 注意事项

### 适用范围
- ✅ v1.0主要针对上市公司
- ✅ 可获取公开财务数据的企业
- ⚠️ 非上市企业需要v2.0接入银行内部系统

### 数据时效性
- 优先使用最新季报/年报
- 重点关注近3个月的变化
- 行业数据尽量使用最新报告
- 建议审贷会前1周内更新

### 提问预测准确性
- 基于财务异常和行业特点生成
- 需要结合具体企业情况调整
- 建议准备额外的应对方案
- 参考历史审贷会提问记录

### 合规要求
- 所有数据来自公开渠道
- 报告仅供内部使用
- 实际授信决策需结合内部评审
- 不能替代尽职调查流程

## 常见问题

**Q: 报告生成需要多长时间？**  
A: 通常3-5分钟，取决于企业的公开信息量和财报复杂度。

**Q: 可以查询哪些企业？**  
A: v1.0主要支持A股、港股、美股上市公司。非上市企业需要v2.0版本。

**Q: 提问预测的准确性如何？**  
A: 基于财务异常和行业特点生成，准确率约70-80%。建议结合历史审贷会记录调整。

**Q: 可以定制行业模板吗？**  
A: 可以。在 `行业特点关注` 部分可以根据具体行业定制关注点。

**Q: 如何提高提问预测的准确性？**  
A: 建议积累历史审贷会提问记录，形成提问库，提高预测准确性。
